# Nova Bible Audit

Audit any Nova Reign image prompt, video prompt, or asset against the Nova Canonical Bible.

## Check:
- Hazel amber golden-brown eyes
- Lean athletic toned build
- Warm medium brown skin
- Long dark wavy/curly hair
- Dog tags visible
- Photorealistic cinematic style
- Vertical 9:16 format
- SFW compliance

## Tattoo Rule (NON-NEGOTIABLE):
LEFT side of image/frame = Nova's right arm with full American flag eagle tattoo sleeve ("1776" and "E Pluribus Unum").
RIGHT side of image/frame = bare arm with zero tattoos.

If tattoo orientation is wrong, return REJECT.
If details are missing but fixable, return FIX.
If all core requirements pass, return PASS.

## Always output:
1. PASS / FIX / REJECT
2. Issues found
3. Exact correction prompt
4. Pre-publish recommendation
