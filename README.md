# Manus Skills: Nova Bootcamp Series 1

This repository contains Manus-compatible skills designed specifically for the Nova Reign Bootcamp video production pipeline. 

These skills ensure consistency, quality, and strict adherence to the Nova Canonical Character Bible across all generated content.

## Available Skills

1. **nova-bible-audit**: Audits any image/video prompt or asset against the locked Nova Bible (physical traits, strict tattoo orientation rule).
2. **brag-nova**: Generates status reports and brag documents for pipeline efficiency.
3. **nova-shorts-package**: Formats the complete metadata and script package for YouTube Shorts using the E-A-S-Y framework.
4. **nova-caption-pack**: Generates tailored captions for YouTube Shorts, TikTok, and Instagram Reels.
5. **nova-production-qa**: Final checklist for audio, visual, and metadata quality before publishing.

## How to Use

Import this repository directly into your Manus Project's Skills tab using the GitHub URL:
`https://github.com/jratdish1/manus-skills`

*Note: This repository contains no secrets, tokens, API keys, or private credentials.*
